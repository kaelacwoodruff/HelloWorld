HTML Lists Reference

Define an HTML list

HTML allows web developers to group a set of related items in lists.

Example

An unordered HTML List:

  • Item
  • Item
  • Item
  • Item

An ordered HTML List:

  1. First Item
  2. Second Item
  3. Third Item
  4. Fourth Item

List Tags

Tag Name Description
<ul> Unordered List These list items will be marked with bullet points by default
<ol> Ordered List These list items will be marked with numbers by default
<li> List Item This element represents a list item of a parent element's list
